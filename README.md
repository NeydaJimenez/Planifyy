# Proyecto: Planify

## Descripción del Problema
Actualmente muchas personas tienen dificultades para organizar sus actividades diarias, ya que suelen olvidar tareas importantes, fechas de entrega, reuniones o compromisos personales y académicos. Esto ocurre por la falta de una herramienta práctica y accesible que permita administrar el tiempo de manera eficiente.

---

## Solución del Problema
La aplicación **Planify** busca solucionar este problema mediante un sistema de organización y agenda digital que ayude a los usuarios a planificar sus actividades, registrar pendientes y recibir recordatorios para mejorar su productividad y administración del tiempo.

---

## Alcance del Proyecto
Planify permitirá a los usuarios registrar tareas, eventos y recordatorios para organizar sus actividades personales, académicas o laborales. El sistema incluirá funciones de creación, edición y eliminación de tareas y eventos, además del envío de recordatorios asociados a cada actividad.

La base de datos almacenará información de usuarios, tareas, recordatorios y eventos, garantizando la relación entre las entidades mediante claves primarias y foráneas.
